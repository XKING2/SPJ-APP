Cara Menjalankan SPJ-APP dileptop Pribadi Anda 
--INI DIJALANKAN HANYA DI LOCALHOST SAJA--

1.Pastikan Anda Memiliki Berapa Software yang dibutuhkan Yaitu ("Laragon/XAMPP,VScode/Code editor,Sqlyog/Php admin") Saya Sarankan untuk menggunakan Sqlyog 
2.Extrack code dari zip yang diberikan 
3.pastikan anda memiliki Database SPJ-APP ('biasanya filenya akan dalam bentuk SQL')
4.Jalankan install database anda ikuti Langkah berikut 
-install sqlyog pada computer anda 
-setelah sqlyog anda terpasang pastikan anda mengaktifkan Xampp Anda atau Laragon anda
-buka sqlyog anda dan klik database dibagian atas head bar
-pilih import lalu pilih execute sql Scrip 
-lalu pilih dimana anda menyimpan Database SPJ-APP
-Tunggu hingga Database berhasil di eksekusi 
5.buka VScode/code Editor anda Lalu buka file SPJ-APP anda
6.Buka Terminal Pada Vscode/Code editor anda Ketikan "PHP artisan Serve"
7.Akan muncul ip Dan port pilih ip itu dan klik control + klik Kanan Pada mouse maka halaman google akan muncul 
8.setelah itu akan muncul halaman login anda bisa memilih beberapa user login 
-1001 -> Kasubag 
-1002 -> Bendahara
-1003 -> user 
-untuk Password semua Sama yaitu password123
Maka Akan Terbuka halaman Dashboard untuk user yang anda loginkan
